Init
---

To install the basics and get started
~~~~
./initStarter.sh
~~~~

Run options
---

To run tests
~~~~
npm test
~~~~

To run locally
~~~~
npm start
~~~~

To dev locally (watching mode)
~~~~
npm run dev
~~~~

Transpile TS files into JS files saved to the **distJs** folder
~~~~
npm run makeJs
~~~~

To login to PCF One
~~~~
./pcfOneLogin.sh
~~~~

To push to PCF
~~~~
./pcfPush.sh
~~~~
